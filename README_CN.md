## ➕添加依赖

- **VastNetStateLayout**

    ```groovy
    implementation 'io.github.sakurajimamaii:VastNetStateLayout:0.0.3'
    ```

- **VastTools**

    ```groovy
    implementation 'io.github.sakurajimamaii:VastTools:0.0.8'
    ```

- **VastAdapter**

    ```groovy
    implementation 'io.github.sakurajimamaii:VastAdapter:0.0.5'
    ```
